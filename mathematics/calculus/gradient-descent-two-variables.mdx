## Function with One Global Minimum

### The Algorithm

For a function $f(x, y)$, starting from initial point $(x_0, y_0)$, update using:

$$x_1 = x_0 - \alpha \frac{\partial f}{\partial x}(x_0, y_0)$$
$$y_1 = y_0 - \alpha \frac{\partial f}{\partial y}(x_0, y_0)$$

where $\alpha > 0$ is the learning rate.

<Note>
The gradient vector $\nabla f = \begin{bmatrix}\frac{\partial f}{\partial x} \\ \frac{\partial f}{\partial y}\end{bmatrix}$ points in the direction of steepest ascent. We move in the opposite direction to minimize.
</Note>

### Implementation

```python
def gradient_descent(dfdx, dfdy, x, y, learning_rate=0.1, num_iterations=100):
    for iteration in range(num_iterations):
        x = x - learning_rate * dfdx(x, y)
        y = y - learning_rate * dfdy(x, y)
    return x, y
```

## Visualizing the Descent Path

Track the path of gradient descent:

```python
def gradient_descent_with_history(dfdx, dfdy, x, y, learning_rate, num_iterations):
    history = [(x, y)]
    
    for i in range(num_iterations):
        x = x - learning_rate * dfdx(x, y)
        y = y - learning_rate * dfdy(x, y)
        history.append((x, y))
    
    return x, y, history

# Run with history tracking
x_min, y_min, path = gradient_descent_with_history(
    dfdx_example_3, dfdy_example_3,
    x_initial=0.5, y_initial=0.6,
    learning_rate=0.25, num_iterations=30
)

# Extract coordinates
path_x = [p[0] for p in path]
path_y = [p[1] for p in path]
path_z = [f_example_3(x, y) for x, y in path]
```

### Contour Plot with Path

```python
# Create contour plot
x = np.linspace(0, 5, 100)

plt.figure(figsize=(10, 8))
plt.contour(X, Y, Z, levels=20, cmap='coolwarm')

plt.plot(path_x, path_y, 'r.-', linewidth=2, markersize=8, label='GD path')

plt.plot(path_x[-1], path_y[-1], 'r*', markersize=15, label='End')

plt.xlabel('x')
plt.ylabel('y')
plt.title('Gradient Descent Path on Contour Plot')
plt.legend()
plt.grid(True, alpha=0.3)

```

## Practical Recommendations

### For Simple Functions (One Minimum)

1. Start with moderate learning rate (0.1-0.3)
2. Monitor convergence with function value plot
3. Adjust learning rate if oscillating or converging too slowly

### For Complex Functions (Multiple Minima)

1. Try multiple random starting points
2. Use smaller learning rates for stability
3. Consider advanced methods (momentum, Adam optimizer)
4. Track best result across all runs

### Debugging Convergence Issues

```python
def diagnose_gradient_descent(dfdx, dfdy, x, y, learning_rate, num_iterations):
    history = {'x': [x], 'y': [y], 'f': [f_example_3(x, y)], 
               'grad_norm': []}
    
    for i in range(num_iterations):
        dx = dfdx(x, y)
        dy = dfdy(x, y)
        grad_norm = np.sqrt(dx**2 + dy**2)
        
        x = x - learning_rate * dx
        y = y - learning_rate * dy
        
        history['x'].append(x)
        history['y'].append(y)
        history['f'].append(f_example_3(x, y))
        history['grad_norm'].append(grad_norm)
    
    # Plot diagnostics
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))
    
    axes[0, 0].plot(history['f'])
    axes[0, 0].set_title('Function Value')
    axes[0, 0].set_xlabel('Iteration')
    axes[0, 0].grid(True)
    
    axes[0, 1].plot(history['grad_norm'])
    axes[0, 1].set_title('Gradient Norm')
    axes[0, 1].set_xlabel('Iteration')
    axes[0, 1].grid(True)
    
    axes[1, 0].plot(history['x'], label='x')
    axes[1, 0].plot(history['y'], label='y')
    axes[1, 0].set_title('Parameter Values')
    axes[1, 0].set_xlabel('Iteration')
    axes[1, 0].legend()
    axes[1, 0].grid(True)
    
    axes[1, 1].plot(history['x'], history['y'])
    axes[1, 1].set_title('Path in Parameter Space')
    axes[1, 1].set_xlabel('x')
    axes[1, 1].set_ylabel('y')
    axes[1, 1].grid(True)
    
    plt.tight_layout()
    plt.show()
    
    return history

history = diagnose_gradient_descent(dfdx_example_3, dfdy_example_3,
                                   x=0.5, y=0.6,
                                   learning_rate=0.25,
                                   num_iterations=30)
```

## Summary

- Gradient descent in 2D follows the same principles as 1D
- The gradient vector points toward steepest ascent; we move opposite
- Learning rate and starting point critically affect convergence
- Multiple minima require multiple random initializations
- Visualization helps understand and debug the optimization process
- These concepts extend to high-dimensional optimization in neural networks
