## Introduction

The perceptron is the simplest neural network model, consisting of a single neuron. In this guide, you'll learn how to implement a perceptron with an activation function for binary classification.

## Binary Classification Problem

**Classification** identifies which category an observation belongs to. **Binary classification** involves only two categories.

### Example: Sentiment Classification

Imagine classifying sentences as "happy" or "angry" based on word counts:
- Count occurrences of words "aack" ($x_1$) and "beep" ($x_2$)
- If $x_2 > x_1$: classify as "angry" (red)
- If $x_2 \leq x_1$: classify as "happy" (blue)

This creates a linear decision boundary: a straight line separating the two classes.

### Visualization

```python
fig, ax = plt.subplots(figsize=(8, 6))
xmin, xmax = -0.2, 1.4
x_line = np.arange(xmin, xmax, 0.1)

# Data points (4 sentences)
ax.scatter(0, 0, color="b", s=100, label="Happy")  # No words
ax.scatter(0, 1, color="r", s=100, label="Angry")  # "Beep!"
ax.scatter(1, 0, color="b", s=100)                 # "Aack?"
ax.scatter(1, 1, color="b", s=100)                 # "Beep aack..."

# Decision boundary: x₂ = x₁ + 0.5
ax.plot(x_line, x_line + 0.5, color="black", linewidth=2, 
        label="Decision boundary")

ax.set_xlim([xmin, xmax])
ax.set_ylim([-0.1, 1.1])
ax.set_xlabel('$x_1$ (aack count)', fontsize=12)
ax.set_ylabel('$x_2$ (beep count)', fontsize=12)
ax.legend()
ax.grid(True, alpha=0.3)
plt.title("Linearly Separable Binary Classification")
plt.show()
```

<Note>
This is called a **linearly separable** problem because a straight line can perfectly separate the two classes.
</Note>

### Finding the Decision Boundary

The line $x_1 - x_2 + 0.5 = 0$ (or $x_2 = x_1 + 0.5$) separates the classes:
- Points where $x_1 - x_2 + 0.5 < 0$ → Red class
- Points where $x_1 - x_2 + 0.5 > 0$ → Blue class

In general form: $w_1x_1 + w_2x_2 + b = 0$

For this example: $w_1 = 1$, $w_2 = -1$, $b = 0.5$

## Single Perceptron Neural Network

### Network Structure

The perceptron model consists of:
- **Input layer**: nodes $x_1$ and $x_2$
- **Weights**: $W = \begin{bmatrix} w_1 & w_2\end{bmatrix}$
- **Bias**: $b$
- **Activation function**: sigmoid $\sigma(z)$

### Forward Propagation

For each training example $x^{(i)} = \begin{bmatrix} x_1^{(i)} & x_2^{(i)}\end{bmatrix}$:

$$
\begin{align}
z^{(i)} &= w_1x_1^{(i)} + w_2x_2^{(i)} + b = Wx^{(i)} + b \\
a^{(i)} &= \sigma(z^{(i)}) = \frac{1}{1+e^{-z^{(i)}}}
\end{align}
$$

### Sigmoid Activation Function

$$
\sigma(z) = \frac{1}{1+e^{-z}}
$$

```python
def sigmoid(z):
    return 1/(1 + np.exp(-z))

print("sigmoid(-2) =", sigmoid(-2))
print("sigmoid(0) =", sigmoid(0))
print("sigmoid(3.5) =", sigmoid(3.5))

# Works element-wise on arrays
print(sigmoid(np.array([-2, 0, 3.5])))
```

### Predictions

Use threshold of 0.5:

$$
\hat{y} = \begin{cases} 
1 & \text{if } a > 0.5 \\ 
0 & \text{otherwise}
\end{cases}
$$

### Vectorized Form

For $m$ training examples in matrix $X$ (shape $2 \times m$):

$$
\begin{align}
Z &= WX + b \\
A &= \sigma(Z)
\end{align}
$$

where $b$ is broadcasted to shape $1 \times m$.

## Loss Function

For binary classification, use **log loss** (cross-entropy):

$$
\mathcal{L}(W, b) = \frac{1}{m}\sum_{i=1}^{m} \left(-y^{(i)}\log(a^{(i)}) - (1-y^{(i)})\log(1- a^{(i)})\right)
$$

where $y^{(i)} \in \{0,1\}$ are true labels and $a^{(i)}$ are predicted probabilities.

## Gradient Descent

### Computing Gradients

Using the chain rule:

$$
\begin{align}
\frac{\partial \mathcal{L}}{\partial W} &= \frac{1}{m}(A - Y)X^T \\
\frac{\partial \mathcal{L}}{\partial b} &= \frac{1}{m}(A - Y)\mathbf{1}
\end{align}
$$

where $(A - Y)$ has shape $1 \times m$, $X^T$ has shape $m \times 2$, and $\mathbf{1}$ is an $m \times 1$ vector of ones.

### Parameter Updates

$$
\begin{align}
W &= W - \alpha \frac{\partial \mathcal{L}}{\partial W} \\
b &= b - \alpha \frac{\partial \mathcal{L}}{\partial b}
\end{align}
$$

where $\alpha$ is the learning rate.

## Implementation

### Dataset

Create a simple dataset:

```python
m = 30

X = np.random.randint(0, 2, (2, m))
Y = np.logical_and(X[0] == 0, X[1] == 1).astype(int).reshape((1, m))

print('Training dataset X (x1, x2 coordinates in columns):')
print(X)
print('Training dataset Y (0: blue, 1: red)')
print(Y)
print(f'Shape of X: {X.shape}')
print(f'Shape of Y: {Y.shape}')
print(f'Number of examples: {X.shape[1]}')
```

### Neural Network Functions

#### 1. Define Network Structure

```python
def layer_sizes(X, Y):
    n_x = X.shape[0]  # Input layer size
    n_y = Y.shape[0]  # Output layer size
    return (n_x, n_y)

(n_x, n_y) = layer_sizes(X, Y)
print(f"Input layer size: n_x = {n_x}")
print(f"Output layer size: n_y = {n_y}")
```

#### 2. Initialize Parameters

```python
def initialize_parameters(n_x, n_y):
    W = np.random.randn(n_y, n_x) * 0.01
    b = np.zeros((n_y, 1))
    
    parameters = {"W": W, "b": b}
    return parameters

parameters = initialize_parameters(n_x, n_y)
print("W =", parameters["W"])
print("b =", parameters["b"])
```

#### 3. Forward Propagation

```python
def forward_propagation(X, parameters):
    W = parameters["W"]
    b = parameters["b"]
    
    Z = np.matmul(W, X) + b
    A = sigmoid(Z)
    
    return A

A = forward_propagation(X, parameters)
print("Output vector A:", A)
```

#### 4. Compute Cost

```python
def compute_cost(A, Y):
    m = Y.shape[1]
    
    logprobs = -np.multiply(np.log(A), Y) - np.multiply(np.log(1 - A), 1 - Y)
    cost = 1/m * np.sum(logprobs)
    
    return cost

print("Initial cost =", compute_cost(A, Y))
```

#### 5. Backward Propagation

```python
def backward_propagation(A, X, Y):
    m = X.shape[1]
    
    dZ = A - Y
    dW = 1/m * np.dot(dZ, X.T)
    db = 1/m * np.sum(dZ, axis=1, keepdims=True)
    
    grads = {"dW": dW, "db": db}
    return grads

grads = backward_propagation(A, X, Y)
print("dW =", grads["dW"])
print("db =", grads["db"])
```

#### 6. Update Parameters

```python
def update_parameters(parameters, grads, learning_rate=1.2):
    W = parameters["W"]
    b = parameters["b"]
    
    dW = grads["dW"]
    db = grads["db"]
    
    W = W - learning_rate * dW
    b = b - learning_rate * db
    
    parameters = {"W": W, "b": b}
    return parameters

parameters_updated = update_parameters(parameters, grads)
print("W updated =", parameters_updated["W"])
print("b updated =", parameters_updated["b"])
```

### Complete Training Loop

```python
def nn_model(X, Y, num_iterations=10, learning_rate=1.2, print_cost=False):
    n_x = layer_sizes(X, Y)[0]
    n_y = layer_sizes(X, Y)[1]
    
    parameters = initialize_parameters(n_x, n_y)
    
    for i in range(num_iterations):
        # Forward propagation
        A = forward_propagation(X, parameters)
        
        # Compute cost
        cost = compute_cost(A, Y)
        
        # Backward propagation
        grads = backward_propagation(A, X, Y)
        
        # Update parameters
        parameters = update_parameters(parameters, grads, learning_rate)
        
        if print_cost:
            print(f"Cost after iteration {i}: {cost}")
    
    return parameters

# Train the model
parameters = nn_model(X, Y, num_iterations=50, learning_rate=1.2, print_cost=True)
print("\nFinal parameters:")
print("W =", parameters["W"])
print("b =", parameters["b"])
```

### Making Predictions

```python
def predict(X, parameters):
    A = forward_propagation(X, parameters)
    predictions = A > 0.5
    return predictions

X_pred = np.array([[1, 1, 0, 0],
                   [0, 1, 0, 1]])
Y_pred = predict(X_pred, parameters)

print(f"Coordinates (in columns):\n{X_pred}")
print(f"Predictions:\n{Y_pred}")
```

### Visualizing Decision Boundary

```python
def plot_decision_boundary(X, Y, parameters):
    W = parameters["W"]
    b = parameters["b"]
    
    fig, ax = plt.subplots(figsize=(10, 8))
    plt.scatter(X[0, :], X[1, :], c=Y, cmap=colors.ListedColormap(['blue', 'red']), 
                s=100, edgecolors='black')
    
    # Plot decision boundary: w₁x₁ + w₂x₂ + b = 0
    x_line = np.arange(np.min(X[0,:])-0.5, np.max(X[0,:])+0.5, 0.1)
    y_line = - W[0,0] / W[0,1] * x_line - b[0,0] / W[0,1]
    ax.plot(x_line, y_line, color="black", linewidth=2, label="Decision boundary")
    
    ax.set_xlabel('$x_1$', fontsize=14)
    ax.set_ylabel('$x_2$', fontsize=14)
    ax.set_title('Perceptron Decision Boundary', fontsize=16)
    ax.legend()
    ax.grid(True, alpha=0.3)
    plt.show()

plot_decision_boundary(X, Y, parameters)
```

## Advantages and Limitations

### Advantages

✅ Simple and interpretable  
✅ Fast to train  
✅ Works well for linearly separable data  
✅ Foundation for understanding neural networks  

### Limitations

❌ Cannot solve non-linearly separable problems (e.g., XOR)  
❌ Limited to binary classification (without modification)  
❌ Sensitive to learning rate  
❌ No hidden layers for feature learning  

## Summary

- The perceptron is a single-neuron neural network for binary classification
- Uses sigmoid activation for smooth, differentiable outputs
- Trained with gradient descent and log loss
- Decision boundary is a straight line (hyperplane in higher dimensions)
- Forms the building block for multi-layer neural networks
- Ideal for understanding basic neural network concepts
