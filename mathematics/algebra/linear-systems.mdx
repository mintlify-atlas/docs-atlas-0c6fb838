By the end of this lesson, you will be able to:

## What is a Linear System?

A **system of linear equations** is a collection of linear equations involving the same variables. For example:

$$
\begin{cases} 
-x_1+3x_2=7, \\ 3x_1+2x_2=1
\end{cases}
$$

To **solve** this system means to find values for $x_1$ and $x_2$ that satisfy both equations simultaneously.

### Singular vs Non-Singular Systems

- **Non-singular**: Has a unique solution
- **Singular**: Has either no solution or infinitely many solutions

## Matrix Representation

The system above can be represented as an **augmented matrix**:

$$
\begin{bmatrix}
-1 & 3 & 7 \\
3 & 2 & 1
\end{bmatrix}
$$

Each row represents an equation:
- First column: coefficients of $x_1$
- Second column: coefficients of $x_2$
- Third column: constants (right-hand side)

We can also separate this into a **coefficient matrix** $A$ and a **constant vector** $b$:

$$
A = \begin{bmatrix}
-1 & 3\\
3 & 2
\end{bmatrix}, \quad
b = \begin{bmatrix}
7 \\
1
\end{bmatrix}
$$

## Creating Matrices in NumPy

```python
import numpy as np

# Coefficient matrix A
A = np.array([
    [-1, 3],
    [3, 2]
], dtype=float)

# Constant vector b
b = np.array([7, 1], dtype=float)

print("Matrix A:")
print(A)
print("\nVector b:")
print(b)
```

Output:
```
Matrix A:
[[-1.  3.]
 [ 3.  2.]]

Vector b:
[7. 1.]
```

### Checking Dimensions

```python
print(f"Shape of A: {A.shape}")  # (2, 2)
print(f"Shape of b: {b.shape}")  # (2,)
```

## Solving Linear Systems

NumPy provides `np.linalg.solve()` to solve systems of the form $Ax = b$:

```python
x = np.linalg.solve(A, b)
print(f"Solution: {x}")
```

Output:
```
Solution: [-1.  2.]
```

This means $x_1 = -1$ and $x_2 = 2$.

### Verifying the Solution

Check that the solution satisfies both equations:

```python
# Equation 1: -x₁ + 3x₂ = 7
eq1 = -x[0] + 3*x[1]
print(f"Equation 1: {eq1} = 7")  # 7.0 = 7

# Equation 2: 3x₁ + 2x₂ = 1
eq2 = 3*x[0] + 2*x[1]
print(f"Equation 2: {eq2} = 1")  # 1.0 = 1
```

Or use matrix multiplication:

```python
result = A @ x  # Matrix-vector multiplication
print(f"A @ x = {result}")
print(f"b = {b}")
print(f"Match: {np.allclose(result, b)}")  # True
```

## Determinants

The **determinant** of a square matrix tells us whether the system has a unique solution.

### Key Property

- **det(A) ≠ 0**: System has a unique solution (non-singular)
- **det(A) = 0**: System has no solution or infinitely many solutions (singular)

### Computing Determinants

```python
d = np.linalg.det(A)
print(f"Determinant of matrix A: {d:.2f}")
```

Output:
```
Determinant of matrix A: -11.00
```

Since the determinant is non-zero, the system has a unique solution.

## Visualizing 2×2 Systems

Each equation in a 2×2 system represents a line. The solution is where the lines intersect.

### Creating the Augmented Matrix

```python
# Stack A and b horizontally
A_system = np.hstack((A, b.reshape((2, 1))))
print(A_system)
```

Output:
```
[[-1.  3.  7.]
 [ 3.  2.  1.]]
```

### Plotting the Lines

```python
import matplotlib.pyplot as plt

def plot_lines(A_system):
    # Extract coefficients
    a1, b1, c1 = A_system[0]
    a2, b2, c2 = A_system[1]
    
    # Create x values
    x = np.linspace(-5, 5, 100)
    
    # Solve for y: ax + by = c → y = (c - ax) / b
    y1 = (c1 - a1*x) / b1
    y2 = (c2 - a2*x) / b2
    
    # Plot
    plt.figure(figsize=(8, 6))
    plt.plot(x, y1, label=f'{a1}x₁ + {b1}x₂ = {c1}')
    plt.plot(x, y2, label=f'{a2}x₁ + {b2}x₂ = {c2}')
    plt.plot(-1, 2, 'ro', markersize=10, label='Solution')
    plt.xlabel('x₁')
    plt.ylabel('x₂')
    plt.legend()
    plt.grid(True)
    plt.axhline(y=0, color='k', linewidth=0.5)
    plt.axvline(x=0, color='k', linewidth=0.5)
    plt.title('Geometric Representation of Linear System')
    plt.show()

plot_lines(A_system)
```

The lines intersect at $(-1, 2)$, confirming our solution!

## Singular Systems: No Solution

Consider this system:

$$
\begin{cases} 
-x_1+3x_2=7, \\ 3x_1-9x_2=1
\end{cases}
$$

```python
A_2 = np.array([
    [-1, 3],
    [3, -9]
], dtype=float)

b_2 = np.array([7, 1], dtype=float)

# Check determinant
d_2 = np.linalg.det(A_2)
print(f"Determinant: {d_2:.2f}")  # 0.00
```

The determinant is zero, so the system is singular.

### Attempting to Solve

```python
try:
    x_2 = np.linalg.solve(A_2, b_2)
except np.linalg.LinAlgError as err:
    print(err)
    # Output: Singular matrix
```

### Visualizing Parallel Lines

```python
A_2_system = np.hstack((A_2, b_2.reshape((2, 1))))
plot_lines(A_2_system)
```

The lines are parallel and never intersect - no solution exists!

## Infinite Solutions

Consider:

$$
\begin{cases} 
-x_1+3x_2=7, \\ 3x_1-9x_2=-21
\end{cases}
$$

```python
b_3 = np.array([7, -21], dtype=float)
A_3_system = np.hstack((A_2, b_3.reshape((2, 1))))

plot_lines(A_3_system)
```

Both equations represent the **same line**, so there are infinitely many solutions!

The solution set is:
$$x_1 = 3x_2 - 7$$

where $x_2$ can be any real number.

## Key Functions

| Function | Purpose | Example |
|----------|---------|----------|
| `np.linalg.solve(A, b)` | Solve $Ax = b$ | `x = np.linalg.solve(A, b)` |
| `np.linalg.det(A)` | Calculate determinant | `d = np.linalg.det(A)` |
| `A @ x` | Matrix-vector product | `result = A @ x` |
| `np.hstack((A, b))` | Stack horizontally | `aug = np.hstack((A, b.reshape(-1,1)))` |
