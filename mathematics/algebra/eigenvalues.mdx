## Introduction

Eigenvalues and eigenvectors are fundamental concepts in linear algebra with applications in machine learning, physics, and data science. In this guide, you'll learn how to compute and interpret them using Python.

## Definition of Eigenvalues and Eigenvectors

For a linear transformation defined by matrix $A$, if there exists a vector $v$ such that:

$$Av=\lambda v$$

Then $v$ is called an **eigenvector** and $\lambda$ is called an **eigenvalue**.

<Note>
Eigenvectors are special vectors that only change in length (by the eigenvalue factor) when the transformation is applied, but maintain their direction.
</Note>

### Important Property

If $v$ is an eigenvector with eigenvalue $\lambda$, then any scaled version $kv$ (where $k \neq 0$) is also an eigenvector with the same eigenvalue:

$$A(kv)=k(Av)=k \lambda v = \lambda (kv)$$

This means for each eigenvalue, there are infinitely many valid eigenvectors, all pointing along the same line.

## Finding Eigenvalues and Eigenvectors with Python

### Setup

```python
import numpy as np
import matplotlib.pyplot as plt
```

### Using np.linalg.eig()

NumPy provides the `np.linalg.eig()` function that returns:
- A vector containing the eigenvalues
- An array with corresponding eigenvectors (one per column)
- Eigenvectors are normalized to have norm 1

```python
A = np.array([[2, 3], [2, 1]])
e1 = np.array([[1], [0]])
e2 = np.array([[0], [1]])

A_eig = np.linalg.eig(A)

print(f"Matrix A:\n{A}")
print(f"\nEigenvalues of matrix A:\n{A_eig[0]}")
print(f"\nEigenvectors of matrix A:\n{A_eig[1]}")
```

### Accessing Individual Eigenvectors

```python
# First eigenvector
v1 = A_eig[1][:, 0]

# Second eigenvector  
v2 = A_eig[1][:, 1]
```

### Visualizing the Transformation

When you apply the transformation to the eigenvectors, they maintain their direction:

```python
# v1 is stretched by a factor of 4
# v2 shows a change of direction (equivalent to factor of -1)
```

<Tip>
Eigenvectors remain parallel to their original direction after transformation, only their length changes by the eigenvalue factor.
</Tip>

## Standard Transformations

### Reflection about y-axis

```python
A_reflection_yaxis = np.array([[-1, 0], [0, 1]])
A_reflection_yaxis_eig = np.linalg.eig(A_reflection_yaxis)

print(f"Matrix A:\n {A_reflection_yaxis}")
print(f"\nEigenvalues of matrix A:\n {A_reflection_yaxis_eig[0]}")
print(f"\nEigenvectors of matrix A:\n {A_reflection_yaxis_eig[1]}")
```

### Shear Transformation

A **shear transformation** displaces points in a fixed direction proportional to their distance from a given line:

$$
A_{\text{shear\_x}} = \begin{pmatrix} 1 & 0.5 \\ 0 & 1 \end{pmatrix}
$$

```python
A_shear_x = np.array([[1, 0.5], [0, 1]])
A_shear_x_eig = np.linalg.eig(A_shear_x)

print(f"Matrix A_shear_x:\n {A_shear_x}")
print(f"\nEigenvalues of matrix A_shear_x:\n {A_shear_x_eig[0]}")
print(f"\nEigenvectors of matrix A_shear_x:\n {A_shear_x_eig[1]}")
```

<Warning>
Some transformations may have complex eigenvalues and eigenvectors. In Python, the imaginary part is indicated with `j` instead of the mathematical notation $i$.
</Warning>

## Special Cases

### Identity Matrix and Uniform Scaling

For the identity matrix, every vector is an eigenvector:

```python
A_identity = np.array([[1, 0], [0, 1]])
A_identity_eig = np.linalg.eig(A_identity)

print(f"Matrix A_identity:\n {A_identity}")
print(f"\nEigenvalues of matrix A_identity:\n {A_identity_eig[0]}")
print(f"\nEigenvectors of matrix A_identity:\n {A_identity_eig[1]}")
```

<Note>
While NumPy only returns two eigenvectors, mathematically all vectors are eigenvectors for the identity matrix. This is why understanding the mathematical concepts behind the code is important.
</Note>

### Scaling in All Directions

For uniform scaling by factor 2:

```python
A_scaling = np.array([[2, 0], [0, 2]])
A_scaling_eig = np.linalg.eig(A_scaling)

print(f"Matrix A_scaling:\n {A_scaling}")
print(f"\nEigenvalues of matrix A_scaling:\n {A_scaling_eig[0]}")
print(f"\nEigenvectors of matrix A_scaling:\n {A_scaling_eig[1]}")
```

### Projection onto x-axis

The projection transformation:

$$
A_{\text{projection}}=\begin{bmatrix}1 & 0 \\ 0 & 0 \end{bmatrix}
$$

```python
A_projection = np.array([[1, 0], [0, 0]])
A_projection_eig = np.linalg.eig(A_projection)

print(f"Matrix A_projection:\n {A_projection}")
print(f"\nEigenvalues of matrix A_projection:\n {A_projection_eig[0]}")
print(f"\nEigenvectors of matrix A_projection:\n {A_projection_eig[1]}")
```

<Note>
This matrix has two real eigenvalues, and one is $\lambda = 0$. This means anything on the y-axis is sent to zero, as it has no component in the x-direction.
</Note>

## Applications

Eigenvalues and eigenvectors are used in:

- **Principal Component Analysis (PCA)**: Dimensionality reduction in machine learning
- **Stability Analysis**: Understanding system behavior over time
- **Quantum Mechanics**: Observable quantities and state vectors
- **Google PageRank**: Ranking web pages
- **Image Compression**: Identifying important features
- **Markov Chains**: Finding steady-state distributions

## Summary

- Eigenvectors maintain their direction under transformation, only scaling by the eigenvalue
- Use `np.linalg.eig()` to compute eigenvalues and eigenvectors in Python
- Different transformations have different numbers of real eigenvectors
- Understanding eigenvalues helps interpret the behavior of linear transformations
- Some matrices (identity, uniform scaling) have all vectors as eigenvectors
