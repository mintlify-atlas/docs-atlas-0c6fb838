## Introduction

A **transformation** is a function from one vector space to another that respects the underlying (linear) structure of each vector space. In this guide, you'll explore linear transformations and see how they're represented using matrices.

## Understanding Transformations

Referring to a specific transformation, you can use a symbol such as $T$. Specifying the spaces containing the input and output vectors, e.g., $\mathbb{R}^2$ and $\mathbb{R}^3$, you can write $T: \mathbb{R}^2 \rightarrow \mathbb{R}^3$.

Transforming vector $v \in \mathbb{R}^2$ into vector $w \in \mathbb{R}^3$ by the transformation $T$, you write $T(v)=w$ and read it as "*T of v equals w*" or "*vector w is an image of vector v with the transformation T*".

### Example Transformation

The following transformation $T: \mathbb{R}^2 \rightarrow \mathbb{R}^3$ has this formula:

$$
T\begin{pmatrix}
          \begin{bmatrix}
           v_1 \\
           v_2
          \end{bmatrix}\end{pmatrix}=
          \begin{bmatrix}
           3v_1 \\
           0 \\
           -2v_2
          \end{bmatrix}
$$

In Python:

```python
import numpy as np

def T(v):
    w = np.zeros((3,1))
    w[0,0] = 3*v[0,0]
    w[2,0] = -2*v[1,0]
    return w

v = np.array([[3], [5]])
w = T(v)

print("Original vector:\n", v, "\n\n Result of the transformation:\n", w)
```

## Linear Transformations

A transformation $T$ is **linear** if these two properties hold for any scalar $k$ and any input vectors $u$ and $v$:

1. $T(kv)=kT(v)$ (scalar multiplication)
2. $T(u+v)=T(u)+T(v)$ (vector addition)

<Note>
Every linear transformation can be carried out by matrix multiplication, and vice versa. This is a key connection between linear transformations and matrix algebra.
</Note>

### Verification Example

For the transformation above:

$$T (kv) =
          T \begin{pmatrix}\begin{bmatrix}
          kv_1 \\
          kv_2
          \end{bmatrix}\end{pmatrix} = 
          \begin{bmatrix}
           3kv_1 \\
           0 \\
           -2kv_2
          \end{bmatrix} =
          k\begin{bmatrix}
           3v_1 \\
           0 \\
           -2v_2
          \end{bmatrix} = 
          kT(v)
$$

Verify in Python:

```python
u = np.array([[1], [-2]])
v = np.array([[2], [4]])
k = 7

print("T(k*v):\n", T(k*v), "\n k*T(v):\n", k*T(v), "\n\n")
print("T(u+v):\n", T(u+v), "\n T(u)+T(v):\n", T(u)+T(v))
```

## Transformations as Matrix Multiplication

Let $L: \mathbb{R}^m \rightarrow \mathbb{R}^n$ be defined by matrix $A$, where $L(v)=Av$.

For the transformation:

$$
L\begin{pmatrix}
          \begin{bmatrix}
           v_1 \\
           v_2
          \end{bmatrix}\end{pmatrix}=
          \begin{bmatrix}
           3v_1 \\
           0 \\
           -2v_2
          \end{bmatrix}=
          \begin{bmatrix}
           3 & 0 \\
           0 & 0 \\
           0 & -2
          \end{bmatrix}
          \begin{bmatrix}
           v_1 \\
           v_2
          \end{bmatrix}
$$

Implementation:

```python
def L(v):
    A = np.array([[3,0], [0,0], [0,-2]])
    print("Transformation matrix:\n", A, "\n")
    w = A @ v
    return w

v = np.array([[3], [5]])
w = L(v)

print("Original vector:\n", v, "\n\n Result of the transformation:\n", w)
```

## Standard Transformations in a Plane

A linear transformation $L: \mathbb{R}^2 \rightarrow \mathbb{R}^2$ can be represented as multiplication of a $2 \times 2$ matrix and a coordinate vector.

### Key Insight

Using standard basis vectors $e_1=\begin{bmatrix}1 \\ 0\end{bmatrix}$ and $e_2=\begin{bmatrix}0 \\ 1\end{bmatrix}$:

$$
A=\begin{bmatrix}L(e_1) & L(e_2)\end{bmatrix}
$$

This is a matrix with columns that are the images of the standard basis vectors.

### Horizontal Scaling (Dilation)

Horizontal scaling by factor 2:

```python
def T_hscaling(v):
    A = np.array([[2,0], [0,1]])
    w = A @ v
    return w

e1 = np.array([[1], [0]])
e2 = np.array([[0], [1]])
```

### Reflection about y-axis

Reflection transformation:

```python
def T_reflection_yaxis(v):
    A = np.array([[-1,0], [0,1]])
    w = A @ v
    return w
```

### Image Transformation Example

Using OpenCV for image transformations:

```python
import cv2
import matplotlib.pyplot as plt

img = cv2.imread('images/leaf_original.png', 0)
plt.imshow(img)
```

Rotate the image 90 degrees clockwise:

```python
image_rotated = cv2.rotate(img, cv2.ROTATE_90_CLOCKWISE)
plt.imshow(image_rotated)
```

Apply shear transformation:

```python
rows, cols = image_rotated.shape
M = np.float32([[1, 0.5, 0], [0, 1, 0], [0, 0, 1]])
image_rotated_sheared = cv2.warpPerspective(image_rotated, M, (int(cols), int(rows)))
plt.imshow(image_rotated_sheared)
```

<Warning>
The order of transformations matters! Generally, $BA \neq AB$ for matrix multiplication.
</Warning>

### Transformation Matrices

Define rotation and shear matrices:

```python
M_rotation_90_clockwise = np.array([[0, 1], [-1, 0]])
M_shear_x = np.array([[1, 0.5], [0, 1]])

print("90 degrees clockwise rotation matrix:\n", M_rotation_90_clockwise)
print("Matrix for the shear along x-axis:\n", M_shear_x)
```

Compare different orders:

```python
print("M_rotation_90_clockwise by M_shear_x:\n", M_rotation_90_clockwise @ M_shear_x)
print("M_shear_x by M_rotation_90_clockwise:\n", M_shear_x @ M_rotation_90_clockwise)
```

## Key Takeaways

- Linear transformations preserve vector addition and scalar multiplication
- Every linear transformation can be represented as matrix multiplication
- Standard basis vectors help visualize transformations
- Order matters when applying multiple transformations
- Applications include computer graphics, data science, and machine learning
