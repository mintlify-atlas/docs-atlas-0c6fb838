    <Step title="Set up your environment">
      Install Python and Jupyter Notebook to run interactive machine learning examples. Jupyter is the default environment used by data scientists and machine learning practitioners worldwide.
      
      ```bash
      pip install jupyter numpy matplotlib
      jupyter notebook
      ```
      
      <Note>We recommend Python 3.8 or higher for full compatibility with modern ML libraries</Note>
    </Step>
    
    <Step title="Explore your first concept">
      Start with supervised learning fundamentals. Learn how machines learn from labeled data to make predictions on new, unseen examples.
      
      ```python
      import numpy as np
      
      # Simple linear regression example
      X = np.array([1, 2, 3, 4, 5])
      y = np.array([2, 4, 6, 8, 10])
      
      # Calculate slope and intercept
      slope = np.sum((X - np.mean(X)) * (y - np.mean(y))) / np.sum((X - np.mean(X))**2)
      intercept = np.mean(y) - slope * np.mean(X)
      
      print(f"y = {slope}x + {intercept}")
      ```
    </Step>
    
    <Step title="Build your first model">
      Apply what you've learned by implementing a complete machine learning model. Follow our hands-on tutorials to train neural networks, optimize cost functions, and make predictions.
